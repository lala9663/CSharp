"두 번째 게임은 미션 수행입니다! 미션 카드에는 다양한 활동이 적혀 있어요. 각 미션을 수행하면 포인트를 받을 수 있고, 가장 많은 포인트를 얻은 사람에게 상품이 주어집니다."

"미션을 수행하면서 다 같이 웃고 즐기는 시간을 보내면 좋겠네요. 자, 첫 번째 미션! '마이크를 잡고 30초 동안 자신만의 춤을 춰주세요!'"
